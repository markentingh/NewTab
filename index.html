Tapestry

A CSS / LESS Dashboard UI framework created for Websilk, a drag & drop web development platform

View on GitHub Download .zip Download .tar.gz

Create an elegant & responsive website & dashboard using Tapestry as your core CSS framework. Simply copy & paste HTML markup from our UI Showcase directly into your web pages.

How it works

  • Add tapestry.css to the head of your website. This file contains a 12 column grid system along with styling for all the UI elements and supports three different sizes of UI elements (small, medium, & large).
  • Choose one of the theme CSS files to add to the head of your website.
  • Load the tapestry.js library at the foot of your website
  • A theme is a list of CSS coloring rules that are applied to the tapestry UI. The Theme Factory LESS file takes a list of HEX color variables from your Theme LESS file and generates all the coloring rules for the UI.

Features

  • Simple Class Naming Structure makes it easy to read your HTML markup.
  • Create your own custom themes by copying one of the Theme LESS files that comes with the framework, then modify a list of HEX colors. It's that easy.
  • Advanced users can modify the Tapestry LESS file to customize the look & feel of the UI.
  • Add coloring rules to the Theme Factory LESS file to alter the way the UI is colored.
